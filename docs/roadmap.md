# Roadmap — Fabric.js (Editor de memes para builders)

## v1 (demo)

Lo que Claude Code debe construir ahora:

- Canvas interactivo con Fabric.js v6
- Imagen preinstalada "Builders Anónimos" cargada por defecto
- Botón para subir imagen propia (reemplaza la imagen base)
- Botón "Añadir texto" que crea un bloque de texto editable en el canvas
- Panel de herramientas: selector de fuente (10 Google Fonts con preview), slider de tamaño, color picker, toggles bold/italic
- Los cambios en el panel se reflejan en tiempo real sobre el texto seleccionado
- Botón de descarga que exporta el canvas como PNG
- Edición directa de texto en el canvas (doble clic para editar)
- Arrastrar, escalar y rotar texto con los controles de Fabric.js
- Layout responsive (desktop: canvas + panel lateral; mobile: canvas + panel abajo)
- Banners fijos: consultoría, newsletter, repositorio
- Sección de stack al final de la página

## v2 (mejoras posibles)

- Permitir múltiples bloques de texto simultáneos con gestión de capas
- Añadir plantillas predefinidas (memes populares del mundo tech)
- Opción de texto con sombra y outline para mejor legibilidad sobre imágenes
- Compartir directamente en redes sociales (Twitter/LinkedIn) vía Web Share API
